# Nexly Dubai

A modern smart business card platform for Dubai professionals.

**Nexly** - The future of networking with NFC-enabled smart business cards.

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Features

- Smart business card catalog
- NFC & QR code enabled cards
- AI-powered design suggestions
- E-commerce flow with cart and checkout
- Order tracking
- User account management

## Tech Stack

- Next.js 15.5.12
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- Firebase
- Genkit AI (Google Gemini)
